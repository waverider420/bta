# BTA = bin to ASCII
This is a simple program that takes in a binary file and creates a textfile consisting of characters repsresending each bit in the given file.

# Usage
./bta binfile
	will produce binfile.txt

- Arguments:
	-b[lock] separates each 8 bits with a space
	-s uses the standard output instead of creating a new text file

# Build
cd into the repo and run make.
